# Code of Conduct

## Our standard

AltmanAI SDK is committed to a professional, respectful, and constructive environment.

We expect contributors to:

- act in good faith
- communicate respectfully
- focus on substance over ego
- welcome useful feedback
- prioritize clarity, safety, and integrity

## Unacceptable behavior

The following behavior is not acceptable:

- harassment or intimidation
- hate speech or discriminatory conduct
- personal attacks
- doxxing or threats
- deliberate disruption of discussion or review
- knowingly deceptive technical claims
- bad-faith security behavior

## Scope

This Code of Conduct applies in project spaces including:

- issues
- pull requests
- code review
- discussions
- project documentation

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this policy.

Serious or repeated violations may result in temporary or permanent exclusion from the project.

## Reporting

For conduct concerns, contact the maintainers through GitHub or the security contact if the matter also involves abuse or platform risk.

Security contact: `support@altmanai.com`
